AWS re:Invent 2025に関する質問に答えてくれるAIエージェント・チャットボットを開発したい。

アプリ名：
re:Inventコンシェルジュ

# 技術スタック

- フロントエンド：Vite + React on AWS Apmlify Gen2
  - Cognitoのマネージド認証画面をつける（メールアドレス＋パスワードのみ）
  - 別アカウントのRoute 53で管理しているドメインで、後から別名を設定します（reinvent.minoruonda.com）
  - ログイン後のチャット画面はシンプルにする。サイドバーなどは設けない。
- バックエンド：Strands Agents on Bedrock AgentCore Runtime
  - Cognito認証のJWTトークンでOAuth認証
  - 以下のツールを利用できる
    - Tavily公式リモートMCP
    - Strands公式のビルトイン「retrieve」MCP
    - re-invent-2025-mcp（ローカルMCP）
  - エージェントのLLMはUS Claude Haiku 4.5を利用
  - 会話履歴を保持して対話を続けられる。ただし画面をリロードすると履歴リセット
- 監視：ユーザーの利用状況は、AgentCore Observabilityを使って、CloudWatch GenAI Observability画面でトレーシングしたい

# 機能

- ユーザーがURLにアクセスすると、Cognitoのデフォルトのサインイン画面が表示される
  - サインイン画面は日本語に翻訳したい
  - 以下のメッセージをサインイン画面に表示したい：「アカウントを作成すれば、誰でもこのアプリを利用できます。登録されたメールアドレスは、アプリ利用時の認証のためだけに利用されます。本アプリの開発者（みのるん）以外にメールアドレスが知られることはありません。また、宣伝などの目的外利用もされません。アカウント作成を行うことで、上記に同意したものとみなします。」
- サインインが成功すると、チャットボットの画面にアクセスできる
  - 右上にサインアウトボタンあり（ユーザーのメールアドレスは表示しない）
- チャットで質問すると、返答してくれる。
  - AIエージェントの思考過程を、ストリーミングでリアルタイム出力してくれる
  - ツールを利用したときは「xxxツールを利用中」と表示され、ツール実行が完了するとそのステータスにチェックマークが付く

# 開発上の注意点

- 期間限定のデモアプリです。複雑な実装はさけ、シンプルで王道な作りを心がけてください。
- 開発者のみのるん自身の学習用途も兼ねています。シンプルで見やすいコードやディレクトリ設計にしてください。
- 新しいサービスを多数使います。AWSナレッジMCPや、Strands MCP、Web検索などで最新情報を徹底収集しながら慎重に実装を進めてください。
- reference ディレクトリに類似案件のソースコードを格納しています。参考にしてください。
- 環境変数やシークレットは、本当に機密度が高いものだけAmplifyやAgentCoreの環境変数に設定して、リスクの低いものはコード内にべた書きしてしまってOKです。
